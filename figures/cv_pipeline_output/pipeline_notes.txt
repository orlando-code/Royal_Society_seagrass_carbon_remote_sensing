CV pipeline: random split vs spatial block CV
============================================

Spatial block CV: fixed sizes 1, 5 km (autocorrelation not used). 
Block sizes (m): 1000, 5000 
CV strategies: Random split + spatial blocks over specified block sizes (m).
Predictors: from raster config (build_covariate_config_from_dir), present in all_extracted_new.rds.
Response: median_carbon_density (from median_carbon_density_100cm, one value per core).

Plots: cv_metrics_by_method_model.png, cv_observed_vs_predicted.png
cv_performance_over_block_sizes.png,
cv_uncertainty_capable_models.png, cv_env_distributions_by_fold.png, cv_spatial_blocks_and_points.png.

Autocorrelation, env_cluster, and distance-buffered CV are commented out.

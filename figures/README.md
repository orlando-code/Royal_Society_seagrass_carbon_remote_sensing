# Figure outputs

- **cv_pipeline_output/** – CV results, permutation pruning, GPR tuning, importance plots, parameter summary.
- **interpolation/** – GPR prediction maps, PDPs, region collage, prediction grid cache.
- **supplement/** – Supplement figures (e.g. region_shapes.png).
- **deprecated/** – Archived outputs; not used by the current pipeline.

All paths are relative to the project root. Generated by `modelling/run_paper_figures.R`.
